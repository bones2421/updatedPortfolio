GifTastic: Pulls from giphy API. Gif generator. Dev process: used HTML and CSS to make layout, then proceeded to incorporate giphy API by grabbing API info from giphy and making the images generated from that info render to the page.

Trivia Game: A quiz style game using timers and JavaScript. Dev process: used HTML and CSS to make the layout then proceeded to incorporate JavaScript to include event listeners and the timer. Used JavaScript to make the buttons work and to keep track of score as well.

Crystal Collector: This is a guessing game. It showcases JavaScript math and comparison. Dev process: used HTML and CSS to create layout. Used math.random methods to pick a number and render it to the page. Used the same method to pick the number the user would select behind the gems. The JavaScript does the math to decide if the number is correct or not and to decide what the value of each gem is.

Liri: A rudimentary SIRI app. This one must be run in node. Dev process: Used node to create a bot that will ask the user questions. Utilized inquirer to get the question format and generate responses for the user to pick. Incorporated yes/no logic to decide how the bot should respond. Assigned values to inputs. Ensured that there were no "maybe" scenarios so the user would not break the app.

Made the layout with HTML and CSS and proceeded to plug in API's. Referenced keys and the necessary credentials to give our site permission to pull information. Plugged in images directly. Hangman: Try to guess the word! Dev process: used HTML and CSS to create the layout then proceeded to plug in the functionality. Used JavaScript to make sounds play and to decide if the user got the letter right or not. The JavaScript also decides the length of the word by picking one at random then figuring out how many characters are in the word. It then refers to the HTML and CSS to verify the placement of the spaces. This verification is done through hard code.
